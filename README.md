# Demonstracao-Z-API
Projeto de demonstração que integra a API da Z-API com o Supabase para envio automatizado de mensagens via WhatsApp. A aplicação consulta uma tabela de contatos armazenada no Supabase (nome e telefone), e para cada contato retornado, envia uma mensagem personalizada de saudação através da Z-API: `"Olá, <nome_contato> tudo bem com você?"`. O projeto utiliza variáveis de ambiente (.env) para configuração segura das credenciais de ambos os serviços, e conta com um sistema de logging que registra o resultado de cada envio tanto no console quanto em arquivo (app.log), tratando erros de timeout, conexão e HTTP de forma individual.

## Preparando ambiente
1. Crie uma conta em https://z-api.io/ e configure sua instância web. Você precisará obter seu token de segurança em https://app.z-api.io/app/security e também o ID e token da sua instância em https://app.z-api.io/app/instances para configurar as variáveis de ambiente corretamente.

2. Crie uma conta no supabase e depois crie um projeto lá. Nas configurações desse projeto (banco de dados), você precisará obter a url supabase e a anon-key para preencher os campos da .env.

3. Ainda no supabase, execute este comando para criar a tabela de contatos:

### Comando de setup da tabela
```
CREATE TABLE contatos (
  id int GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  telefone_contato varchar(13) NOT NULL,
  nome_contato text
);
```

4. Ao fim, preencha a tabela com os dados de contatos que você deseja enviar a mensagem.

## Como iniciar
1. Crie um ambiente virtual:
```bash
python -m venv .venv
```
2. Ative o ambiente virtual:
```bash
.venv\Scripts\activate
```
3. Instale as dependências:
```bash
pip install -r requirements.txt
```
4. Crie um arquivo .env com as seguintes variáveis (igual há no arquivo .env.example):
```
ZAPI_INSTANCE_ID=seu_instance_id
ZAPI_TOKEN=seu_token
ZAPI_CLIENT_TOKEN=seu_client_token
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=sua-anon-key
```
5. Execute o projeto:
```bash
python main.py
```
